Help Section
This is a help website for discord bot created by BanKai#2366.
Command Uses
hello To test if bot is online or not.
++quote Give a random quote
++meme Gives a random meme
++test A test template of embed.
++ban Cannot ban.
++kick Kick command.
++help To get to this website for list of commands.
++slap Slaps the user.
++pfp Gives the Avatar of the mentioned user. If no one is mentioned gives the Avatar of the User who used the command.
++gif New Search any gif on tenor using given query with this command. Example: ++gif cat will give u random cat gif.
++poke Gives the details about that Pokemon. Example: ++poke pikachu
